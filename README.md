# qq群设置群头衔

> 必须bot是群主才能使用咪

## 3种使用方式

- qst

    然后bot会让你输入新头衔，发送你的新头衔，bot会给你改头衔

- qst \<touxian:str>

    修改自己的头衔

- qst \<touxian:str> \<user:at>

    修改指定用户的头衔
